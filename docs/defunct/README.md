# Defunct Documentation

This folder contains the original markdown documentation files that have been migrated to the Docusaurus documentation framework.

## Migration Date
January 2025

## Status
These files are preserved for historical reference but are no longer actively maintained. All content has been migrated to the new Docusaurus structure in `docs/docs/`.

## Structure
The original directory structure has been preserved:
- `guides/` - User guides and tutorials
- `reference/` - Technical reference documentation
- `reports/` - Analysis and implementation reports
- `research/` - Research documents
- `roadmap/` - Roadmap and planning documents
- `design/` - Design mockups and specifications
- Root level files - Various planning and enhancement documents

## Active Documentation
For the current, actively maintained documentation, see:
- `docs/docs/` - Docusaurus documentation source files
- `docs/sidebars.js` - Documentation navigation structure
- `docs/docusaurus.config.js` - Docusaurus configuration

## Note
These files may contain outdated information. Always refer to the Docusaurus documentation for the latest information.
